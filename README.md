Bilibili@1812z 求投币支持
原项目:https://github.com/updeshxp/​scrcp​​-android

# 使用方法
1.手机端使用adbwifi开启无线adb，并把分辨率改手表分辨率x2，dpi建议240-280，列如手表是368x468，手机就改736x936
2.手表端输入ip地址链接
3.手机授权adb请求
